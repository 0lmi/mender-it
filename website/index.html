Let's secure embedded devices - Let's Mender it (!t)

Everyone knows that Linux is cool and security is must have. Everybody likes open sourcse and free public services. Mender!t is inspired by great free service Let's Encrypt and cool open source product Mender, focused on bringing and improving security into embedded world. So, let's make securing embedded devices a little bit easier by making some automation around Mender.

A little bit background

About 3 years ago I worked on a software was running on devides similar to Raspberry Pi and I started looking into firmware updater, but didn't find something I could use, so, now I'm extremely happy that Mender exists. May this year Mender started their SaaS platform which is b2b service, but would be cool to have something for personal usage in smaller scale. Additionally would be great to make public service to configure cloud infrastructure and run your personal Mender server automatically.

Nearest plans

First goal is to run personal Mender server. It can be done by modifying Mender demo server start scripts to use persistent storage and run it in a cloud.

Everyone are very welcome to join and participate in the project, all code is available on Github.
